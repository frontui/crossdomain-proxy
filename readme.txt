# 注意

* index.html
  系统demo模板，必须引进bank.css与autoHeight.js文件。

* proxy.html
  iframe自适应高度调整跨域代理文件，存放地址可访问url请提供给钱途技术人员。

* autoHeight.js
  调整iframe的高度，修改index.html的iframe id后请同步修改此文件的获取iframe的id。

* bank.css
  网页顶部样式。


* 说明
  1.E+账户系统页面和收银台页面，都需用index.html框架页进行嵌套；
  2.index.html中iframe的嵌套src地址，需要根据不同的E+环境进行调整。
